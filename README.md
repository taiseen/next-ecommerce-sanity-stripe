## Package Installed
1. yarn add @babel/core

3. yarn add @sanity/client

5. yarn add @stripe/stripe-js 
6. yarn add stripe
7. yarn add react-icons
8. yarn add react-hot-toast
9. yarn add canvas-confetti
10. yarn add next-sanity-image
